# Placeholder Assets

The following placeholder files should be created in the `assets/` directory:

## Required Assets

1. **icon.png** (1024x1024) - App icon
2. **splash.png** (2048x2048) - Splash screen image
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon
4. **favicon.png** (48x48) - Web favicon

For now, you can use any blue/black gradient circular images or the Spark orb concept.

## Creating Placeholder Assets

You can use online tools or design software to create these:
- Use a blue to black gradient
- Circular shape
- Add glow effects
- Match the Spark branding

Alternatively, Expo will use default assets if these files are not present during development.
